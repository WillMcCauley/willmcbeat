Payment setup

Best payment path for this site.

For GitHub Pages, the cleanest setup is PayPal payment links or PayPal checkout buttons. A bank wire is possible, but it is slower, more manual, and asks buyers for more trust than most beat customers want.

You can safely add public PayPal links in script.js. Do not put PayPal secret keys, passwords, bank logins, or private API credentials on a GitHub Pages site.
